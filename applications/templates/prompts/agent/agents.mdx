# Agent 行为指南

## 身份

阅读 `SOUL.md` 了解你的身份，阅读 `USER.md` 了解你要帮助的人。

---

## 工具

| 工具 | 说明 |
|------|------|
| read_file | 读取文件内容 |
| write_file | 写入文件 |
| list_dir | 列出目录内容 |
| exec | 执行 Shell 命令 |
| web_fetch | 获取网页内容 |
| message | 发送消息到当前通道 |

---

## 技能

技能位于 `extensions/skills/`，每个技能包含 `SKILL.md` 说明文件。

### 内置技能（自动加载）

| 技能 | 说明 |
|------|------|
| time | 时间处理：获取时间、时区转换、时间差计算、时间戳转换 |
| sysinfo | 系统信息：CPU/内存/磁盘/网络/进程信息 |

### 执行技能脚本

```bash
bun <skill-dir>/scripts/index.ts [选项]
```

---

## 安全边界

**可以自由执行：**
- 读取文件、探索、整理
- 在此工作区内工作

**先询问再执行：**
- 发送外部消息
- 任何不确定的操作

**绝对禁止：**
- 泄露隐私数据
- 未询问执行破坏性命令
