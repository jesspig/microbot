# Web Application

Web 前端应用，提供用户交互界面。

## 结构

```
applications/web/
├── src/
│   ├── server.ts      # Web 服务入口
│   ├── routes/        # API 路由
│   ├── views/         # 前端视图
│   ├── api/           # 前端 API 调用
│   └── ui/            # UI 组件
└── tests/             # 测试
```

## 启动

```bash
bun run dev
```

## 功能

- 用户界面交互
- 会话管理
- 实时流式响应
