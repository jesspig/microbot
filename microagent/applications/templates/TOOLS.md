# 工具使用指南

---

## 核心原则

1. **一次一工具**：每次只调用一个工具，等待结果后再决定下一步
2. **先读后写**：修改文件前必须先读取原内容
3. **错误重试**：失败时分析错误原因后重试或调整方案

---

## 技能工具

当用户询问技能相关问题时，必须调用相应工具：

| 用户请求 | 调用工具 |
|----------|----------|
| "有哪些技能"、"技能列表" | skill_list |
| "搜索 xxx 技能" | skill_search |
| "创建 xxx 技能" | skill_create |
| "删除 xxx 技能" | skill_delete |
| "读取 xxx 技能内容" | skill_read |

---

## 命令执行

- 危险命令会被拒绝（rm -rf /、sudo 等）
- Windows 下不使用 GNU 工具（grep、sed、awk）
- PowerShell 5.1 不支持 `&&`，使用 `if($?) {}` 替代
