# v0.2.1

**发布日期**: 2026-02-27

项目重命名与代码清理版本。

## 架构重构

- **项目重命名** - `microbot` 重命名为 `micro-agent`：
  - 包命名空间 `@microbot/*` → `@micro-agent/*`
  - CLI 命令更新为 `micro-agent`
  - 配置目录更新为 `~/.micro-agent/`
  - 扩展字段 `microbot` → `microAgent`
- **类型系统统一** - 将 `providers/base.ts` 类型迁移到 `@micro-agent/types`：
  - 新增 `LLMMessage`、`LLMResponse`、`ToolCall` 类型
  - 新增 `ContentPart` 多模态类型
  - 移除废弃的 `session.ts` 类型文件
- **配置简化** - 统一执行器参数：
  - 移除 `AgentConfigSchema.maxToolIterations`
  - 使用 `executor.maxIterations` 替代

## 改进

- **npm 发布工作流** - 新增 `.github/workflows/publish.yml`，推送 master 自动发布
- **文档更新** - 明确声明 Bun 运行时要求，不支持 Node.js
- **命名规范** - `MemoryStore` 重命名为 `KVMemoryStore`，语义更清晰

## 移除

- **A2A 客户端** - 移除 `packages/providers/src/a2a/` 整个目录
- **模型配置服务** - 移除 `packages/providers/src/model-config.ts`
- **ReAct 提示词** - 移除废弃的 `apps/prompts/react.ts` 和 `react.md`
- **未使用导出** - 清理 SDK 和运行时中未使用的导出函数
