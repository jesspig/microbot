# 更新日志

## v0.1.0

**发布日期**: 2026-02-19

首个内测版本，包含核心功能实现。

### 新增功能

#### 核心架构

- **依赖注入容器** - 轻量级 DI 容器，支持瞬态和单例模式
- **事件总线** - 类型安全的事件发布订阅系统
- **钩子系统** - 前置/后置钩子支持
- **中间件管道** - 可组合的中间件处理链

#### LLM 支持

- **Provider 抽象** - 统一的 LLM 提供商接口
- **OpenAI 兼容** - 支持 OpenAI、DeepSeek、Ollama 等
- **智能路由** - 根据任务复杂度自动选择模型
- **故障转移** - 主 Provider 失败时自动切换备用

#### Agent 系统

- **ReAct 循环** - 思考-行动模式的智能代理
- **上下文构建** - 自动加载记忆、技能、历史
- **工具调用** - 支持多轮工具调用
- **子代理** - 任务委托与并行执行

#### 工具系统

- **文件操作** - read_file, write_file, list_directory, glob, search_file_content
- **Shell 执行** - run_shell_command
- **Web 工具** - web_search, web_fetch
- **消息工具** - send_message

#### 技能系统

- **SKILL.md 规范** - Markdown 格式定义技能
- **渐进式加载** - 按需加载技能内容
- **内置技能** - time, sysinfo

#### 通道系统

- **飞书通道** - WebSocket 长连接，支持消息收发

#### 存储系统

- **会话存储** - JSONL 格式，支持历史查询
- **记忆存储** - SQLite 数据库，支持关键词搜索
- **定时任务存储** - SQLite 持久化

#### 服务系统

- **定时任务服务** - 支持 cron 表达式
- **心跳服务** - 定期健康检查

### 文档

- **VitePress 文档站** - 完整的在线文档
- **Mermaid 图表** - 架构和流程可视化
- **GitHub Pages 部署** - 自动化文档发布

### 技术栈

- **运行时**: Bun 1.3.9+
- **语言**: TypeScript 5.9.3
- **验证**: Zod
- **LLM SDK**: Vercel AI SDK
- **日志**: @logtape/logtape
- **文档**: VitePress

### 已知限制

- 仅支持飞书通道（QQ、钉钉、企业微信待实现）
- 长期记忆暂不支持向量嵌入
- 热插拔机制待完善
